### ✅ 需要的 (6 个核心)
| 合约 | 原因 |
|------|------|
| IdentityRegistry | Agent NFT/注册 |
| SubscriptionManager v3 | 订阅+试用托管+分账 (ReentrancyGuard) |
| A2AProtocolRegistry | A2A 任务+X402 自动扣款 |
| ReputationRegistry | 评分/评价 |
| ConfigurationRegistry | 链上 Agent 配置 |
| MultiEndpointRegistry | 多端点路由 (未来必用) |

### ❌ 不需要的 (6 个)
| 合约 | 原因 |
|------|------|
| PaymentGateway | X402+SM v3 已替代 |
| AgentFactory | IR 直接注册，不走模板工厂 |
| AgentWallet | 无 per-agent 钱包需求 |
| TokenPriceOracle | 无多币种转换 |
| ValidationRegistry | 无链上验证需求 |
| BaseReputationRegistry | 抽象合约 |

### 关键决策
- **PaymentGateway vs SM v3**: SM v3 自闭环（订阅+试用托管+分账），PaymentGateway 是早期泛化支付方案。AgentX 只用 SM v3
- **MultiEndpoint**: 当前 IPFS metadata 存单个 MCP URL，但未来多地区/故障转移需要。保留合约+SDK
- **AgentX402**: 已实现自动扣款，替代 PaymentGateway 的 A2A 支付功能
